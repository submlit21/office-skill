# Office Skill Examples

This directory contains usage examples for the Office Skill.

## Examples

1. [basic_template_management.md](basic_template_management.md) - Basic template management workflow
2. [generate_document_from_template.md](generate_document_from_template.md) - Generate a document from an existing template
3. [analyze_document.md](analyze_document.md) - Analyze a document's structure
4. [convert_to_markdown.md](convert_to_markdown.md) - Convert an Office document to Markdown

## Getting Started

All examples assume you have:
1. Installed `office-skill` with all dependencies: `pip install -e .`
2. Have `libreoffice` available for conversion
3. Have a working `cli-anything-libreoffice` installation
